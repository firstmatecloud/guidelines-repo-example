# Guidelines Repo
This repo contains company-wide guidelines to be followed by all developers and enforced by the FirstMate application.



A company guideline repo should follow this structure:



- **GUIDELINES.md** 
- **Category 1** 
  - **GUIDELINES.md**  
  - **Subcategory 1** 
    - **example** 
    - **GUIDELINES.md**
  - **Subcategory 2**
    - **example** 
    - **GUIDELINES.md** 
- **Category 2** 
  - **Subcategory 1** 
      - **example** 
      - **GUIDELINES.md**



For example:


- **GUIDELINES.md** 
- **Microservices** 
  - **GUIDELINES.md**  
  - **nodejs** 
    - **example** 
    - **GUIDELINES.md**
  - **python**
    - **example** 
    - **GUIDELINES.md** 
- **infra** 
  - **terragrunt** 
      - **example** 
      - **GUIDELINES.md**
